📊 Sales Report Dashboard
An interactive Sales Report Dashboard built using Microsoft Power BI, providing visual insights into sales performance across regions, time periods, and customer segments.

🔍 Overview
This dashboard helps analyze and monitor sales data through dynamic visualizations. It enables quick decision-making by presenting key metrics in a clean, interactive format.

📌 Features

🗺️ Sales by Region/City — Visual breakdown of sales performance across different geographical areas
📅 Monthly/Yearly Trends — Time-based analysis to track growth, seasonality, and performance over time
👥 Customer Analysis — Insights into customer behavior, top customers, and purchase patterns

Explore the dashboard!


📷 Dashboard Sections

Overview Page — High-level KPIs and summary metrics
Regional Analysis — Sales map and city-wise breakdown
Trend Analysis — Monthly and yearly sales charts
Customer Insights — Top customers and segment analysis
